&nbsp;

# **🎯 PROMPT CORRIGÉ (VERSION PORTAIL + PAGES RESTO)**

&nbsp;

&nbsp;

Crée une application SaaS de commande en ligne pour restaurants (kebab, tacos, fast-food), basée sur un **portail central + pages individuelles par restaurant**.

&nbsp;

L'objectif n'est PAS de créer une marketplace avec des centaines de restaurants affichés ensemble, mais un système où **chaque restaurant possède sa propre page dédiée, optimisée pour la conversion et le SEO**.

&nbsp;

---

&nbsp;

&nbsp;

## **🧠 CONCEPT GLOBAL**

&nbsp;

&nbsp;

- Domaine principal :
  → exemple : [commande-maintenant.com](http://commande-maintenant.com)
- Structure :
  &nbsp;
  - Une **landing page simple**
  - Des **pages individuelles type : /nom-du-restaurant-ville**
  &nbsp;

&nbsp;

&nbsp;

👉 Chaque restaurant a sa **page unique**, comme un mini-site.

&nbsp;

---

&nbsp;

&nbsp;

## **🍔 PAGE RESTAURANT (ÉLÉMENT CENTRAL)**

&nbsp;

&nbsp;

&nbsp;

### **Objectif :**

&nbsp;

&nbsp;

Transformer un visiteur en client en moins de 10 secondes.

&nbsp;

&nbsp;

### **Contenu :**

&nbsp;

&nbsp;

- Hero section :
  &nbsp;
  - Nom du restaurant
  - Note + avis
  - Adresse + horaires
  - Bouton CTA principal : **"Commander maintenant"**
  &nbsp;
- Menu :
  &nbsp;
  - Catégories (Sandwichs, Assiettes, Boissons...)
  - Produits avec :
    &nbsp;
    - Nom
    - Prix
    - Options (sauces, suppléments)
    &nbsp;
  - Ajout au panier instantané
  &nbsp;
- Panier sticky :
  &nbsp;
  - Toujours visible
  - Mise à jour en temps réel
  &nbsp;
- Tunnel de commande :
  &nbsp;
  - Ultra court (max 2-3 étapes)
  - Click & collect ou livraison
  - Paiement simple
  &nbsp;

&nbsp;

&nbsp;

---

&nbsp;

&nbsp;

## **🧑‍🍳 BACK-OFFICE RESTAURATEUR**

&nbsp;

&nbsp;

&nbsp;

### **Objectif :**

&nbsp;

&nbsp;

Permettre à un restaurateur de gérer SA page sans effort.

&nbsp;

Fonctionnalités :

&nbsp;

- Modifier :
  &nbsp;
  - Nom
  - Horaires
  - Menu
  - Prix
  - Disponibilité produits
  &nbsp;
- Voir les commandes :
  &nbsp;
  - En temps réel
  - Statuts simples :
    &nbsp;
    - Nouvelle
    - En préparation
    - Prête
    &nbsp;
  &nbsp;
- Activer / désactiver la prise de commande

&nbsp;

&nbsp;

---

&nbsp;

&nbsp;

## **🔗 STRATÉGIE D'ACQUISITION (TRÈS IMPORTANT)**

&nbsp;

&nbsp;

- Chaque restaurant reçoit son lien unique :
  → [commande-maintenant.com/kebab-metz-centre](http://commande-maintenant.com/kebab-metz-centre)
- Le restaurateur :
  &nbsp;
  - Met ce lien sur sa fiche Google Business Profile
  - Le partage sur Instagram / TikTok
  - L'utilise comme site officiel
  &nbsp;

&nbsp;

&nbsp;

👉 L'application agit comme un **site web clé en main + outil de commande**

&nbsp;

---

&nbsp;

&nbsp;

## **🚫 IMPORTANT : PAS UNE MARKETPLACE**

&nbsp;

&nbsp;

- Ne PAS afficher une liste massive de restaurants
- Pas de "catalogue global"
- Chaque page doit être indépendante

&nbsp;

&nbsp;

👉 Option :

&nbsp;

- Une homepage très simple avec recherche
- Ou redirection directe vers pages restaurants

&nbsp;

&nbsp;

&nbsp;

## **🎨 DESIGN**

&nbsp;

&nbsp;

- Style **Apple-like** :
  &nbsp;
  - Minimaliste
  - Ultra lisible
  - Beaucoup d'espace blanc
  - Typo Inter / SF Pro
  &nbsp;
- UX :
  &nbsp;
  - Commande en 2-3 clics max
  - Zéro friction
